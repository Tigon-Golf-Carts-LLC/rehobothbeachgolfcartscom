## RESPONSIBLE VULNERABILITY DISCLOSURE
We appreciate responsible disclosure of security vulnerabilities.
If you discover a security issue, please contact us immediately.

### Reporting Process
1. Email security concerns to: rehobothbeachgolfcarts@gmail.com
2. Include detailed information about the vulnerability
3. Allow reasonable time for investigation and resolution
4. Avoid public disclosure until the issue is resolved

### What to Include in Reports
- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact assessment
- Any relevant screenshots or proof-of-concept
- Your contact information for follow-up
